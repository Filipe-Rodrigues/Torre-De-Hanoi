This Github project will handle almost everything involving Hanoi Tower! Our objective is to create various versions of this game for fun/study reasons, where we seek to enhance the performance or the graphical quality of the subsequent applications. Allowed Java and C++ projects only.


Projeto do Github para Torre de Hanoi! O objetivo desse projeto é criar diversas versões desse game por diversão/estudo, onde sempre que possível, poderemos procurar por melhorar o desempenho ou a qualidade gráfica das aplicações. Aceita-se projetos em Java e C++ apenas.
